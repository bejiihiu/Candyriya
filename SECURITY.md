# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest stable build | ✅ |
| Beta builds | ⚠️ Best effort |
| Older builds | ❌ |

## Reporting a Vulnerability

If you find a security vulnerability in Candyriya, please report it responsibly:

1. **Do NOT open a public issue.**
2. Email or DM on Telegram (devfolia.t.me) with details.
3. Include steps to reproduce if possible.

We will respond within 48 hours and work on a fix. Credit will be given unless you prefer anonymity.

## Scope

Candyriya is a fork of Arclight. Vulnerabilities in upstream Arclight code should be reported to [IzzelAliz](https://github.com/IzzelAliz/Arclight) as well.
